¿Cómo Funciona?

Detección de Incendios y Activación del Sistema

El sistema está diseñado para detectar incendios de manera autónoma mediante sensores de flama. Cuando los sensores de flama detectan la presencia de fuego, envían una señal al microcontrolador Arduino. Este sistema procesa la información y, como medida de seguridad, espera 3 segundos para confirmar que el fuego sigue presente y no es una falsa alarma.

Si la señal de fuego persiste después de este breve retraso, Arduino envía una señal al módulo relay, el cual activa la bomba de agua durante 3 segundos, proporcionando la acción necesaria para extinguir el fuego de manera inmediata.

Detección de Gases Tóxicos

Además de la detección de fuego, el sistema está equipado con un sensor de gas MQ2, que tiene la capacidad de detectar gases peligrosos en el ambiente, como el monóxido de carbono y otros gases tóxicos. Si se detecta una concentración peligrosa de gas, el sensor activa un LED de alerta para informar a los usuarios de la presencia de una situación de riesgo.

Objetivos

Objetivo General

Desarrollar un vehículo autónomo capaz de detectar incendios en lugares de difícil acceso y activarse automáticamente para extinguir las llamas de manera efectiva.

Objetivos Específicos

  • Integrar sensores de flama y gas para detectar fuego y gases tóxicos de forma precisa.
  • Desarrollar un sistema de control mediante Arduino para activar la bomba de agua al detectar fuego, optimizando la respuesta del sistema.
  • Implementar un sistema de alerta para notificar al usuario mediante SMS o llamada, en caso de que se detecten incendios o gases peligrosos.
  • Realizar pruebas exhaustivas en diferentes escenarios, asegurando que el sistema funcione correctamente en condiciones reales y de emergencia.
  • Fomentar la colaboración y el trabajo en equipo durante todas las etapas del desarrollo, para garantizar la calidad y efectividad del prototipo final.

Arduino Uno

Es el cerebro del prototipo. Esta placa microcontroladora gestiona los datos de los sensores y controla el encendido del sistema de extinción (bomba de agua). Se utiliza debido a su accesibilidad y flexibilidad en proyectos de electrónica

Sensores de flama

Son los encargados de detectar la presencia de fuego. Funcionan emitiendo radiación infrarroja que las llamas generan. Cuando cualquiera de estos sensores detecta fuego, envía una señal a la placa Arduino para activar el sistema de extinción.

Sensor MQ2

Este sensor detecta gases peligrosos como monóxido de carbono, metano, etanol y humo. Cuando se detecta un gas tóxico, el sensor activa un LED que alerta sobre la presencia de gases peligrosos. Este sensor agrega una capa extra de seguridad al sistema.

Bomba de agua

La bomba se activa mediante un módulo relay cuando se detecta fuego. Está diseñada para rociar agua en el área afectada durante 3 segundos, tras lo cual se apaga automáticamente. Esta función permite la extinción rápida del fuego.

Modulo reley

Controla el flujo de corriente hacia la bomba de agua. Recibe la señal de Arduino y se activa 3 segundos después de que se detecta fuego, permitiendo que la bomba se encienda durante 3 segundos para apagar las llamas.

Servomotor

Utilizado para realizar movimientos mecánicos en el prototipo, como la dirección del chorro de agua o ajustes en la estructura. El servomotor proporciona la capacidad de realizar movimientos específicos y controlados dentro del sistema de extinción.
